## Decisions

- The MVP implementation stays intentionally narrow: it protects the cultivation jade-talisman/lingyinfu repeated-discovery loop first, while leaving broader thread extraction for future expansion.
- Provider failures should keep the playtest session alive with marked local fallback, matching the existing README/technical-standard promise. Fallback events must still honor story contracts.
