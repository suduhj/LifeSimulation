# Talent System

The authoritative full rules live in `docs/talent-and-scaling-rules.md` and `docs/potential-manifestation-exposure-rules.md`.

## Draw Rules

- Draw 5 talents.
- Keep 3 talents.
- Talent pools include universal talents and world-specific talents.

## Rarity Probability

| Rarity | Probability |
|---|---:|
| Common | 45% |
| Fine | 28% |
| Rare | 16% |
| Epic | 7% |
| Legendary | 3% |
| Mythic | 1% |

## Talent Effect Types

Talents may:

- Add or reduce attributes.
- Change available routes.
- Trigger hidden events.
- Add side effects.
- Modify AI narrative assumptions.
- Unlock world-specific opportunities.
- Add growth multipliers.
- Modify breakthrough results.

Mythic talents should define manifestation type: immediate manifestation, stage manifestation, conditional awakening, or hidden destiny.
