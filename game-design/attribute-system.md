# Attribute System

The authoritative full rules live in `docs/attribute-and-npc-rules.md` and `docs/potential-manifestation-exposure-rules.md`.

## Core Attributes

The game has exactly 5 core attributes:

- Appearance
- Intelligence
- Constitution
- Family Background
- Luck

Do not use Spirit, Charisma, or Combat as core attributes.

## Derived Capabilities

- Stress resilience: Constitution + Intelligence + experience/state history.
- Social charm: Appearance + Family Background + Intelligence + reputation.
- Combat ability: Constitution + skills + profession + realm + equipment.

## Three Attribute Layers

Each attribute tracks:

- Potential value: future height implied by points, talents, identity, and growth systems.
- Manifested value: current age-appropriate expression.
- Exposure value: how much the outside world notices abnormality.

AI must narrate from manifested value, not full potential, especially during birth and childhood.

## Attribute Change Display

Attribute changes must show change amount, current value, source type, temporary/permanent status, and expected recovery condition if recoverable.

Example:

```text
Constitution -2 (current 5, source: serious injury, temporary, recovery: treatment or rest)
```
