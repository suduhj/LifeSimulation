# 克苏鲁生活世界

## 世界定位

克苏鲁生活世界是 MVP 三个世界之一。它是一个架空现代或近现代都市社会：表面和现实高度相似，普通人可以上学、工作、恋爱、结婚、创业、养老；但世界底层隐藏着旧日存在、异常事件、污染、梦境、禁忌知识、秘密组织和不可名状的真相。

这个世界不是强制调查员游戏。玩家角色可以追求真相，也可以逃避真相，过普通人生、谈恋爱、赚钱、加入组织、追求权力、利用旧日传承、封印或斩杀旧日存在、成为类克苏鲁存在、拯救世界、毁灭世界，或走向更怪异的结局。

## 人生模拟重点

克苏鲁生活世界的重点不是从出生开始打怪，而是普通人生与宇宙真相之间的长期拉扯。

AI 写作时必须保留：

- 家庭、学校、职业、恋爱、婚姻、朋友、社会地位等普通人生路线。
- 真相可以慢慢靠近，也可以被玩家角色主动回避。
- 高智力能帮助理解禁忌知识，但也可能增加理智压力和污染风险。
- 高运气不是绝对安全，而是更容易遇到转机，也更容易被卷入稀有危险。
- 玩家探索城市、询问他人、上网搜索、使用资源或仪式都可能获得线索，也可能被注意、误导或卷入险地。

## 世界专属进度

| 进度 | 含义 |
|---|---|
| 真相揭露 | 玩家角色知道多少世界真相。越高不一定越好。 |
| 理智压力 | 接触异常信息后的心理压力和现实感动摇。 |
| 污染/同化 | 被旧日、诡异、梦境、血脉或仪式改变的程度。 |
| 神秘接触 | 与异常存在、组织、禁书、仪式、梦境的接触深度。 |
| 社会正常度 | 玩家角色还能否维持普通人生、身份和关系。 |
| 个人目标 | 玩家角色自己的生活目标、野心、执念或逃避方向。 |

## 公开程度

默认采用：异常半公开，真相高度封锁。

普通人知道世界偶尔有怪事，但不知道旧日、污染、禁忌知识和真正异常结构。官方知道更多，但对外不公开承认。

默认公开等级是 2：传闻广泛，但官方否认。

异常常被解释为疾病、事故、犯罪、谣言、宗教事件、精神问题、网络怪谈、自然灾害、恐怖袭击或实验事故。

## 城市舞台

MVP 使用随机城市舞台，而不是一个固定默认城市。

城市舞台决定本局更容易出现的社会结构、NPC 身份锚点、异常解释方式、组织存在感、真相密度和普通生活空间。

MVP 城市类型包括：

- 架空沿海大都市
- 旧城与周边小镇
- 现代大学城
- 财团控制城市
- 边境工业城市
- 安静郊区

程序读取细则见 `worlds/cthulhu/city-archetypes.json`。

## 核心路线

- 普通人生线：尽量维持正常生活，恋爱、事业、家庭、养老。
- 真相追寻线：通过学习、探索、组织、梦境或事件逐步接触真相。
- 秘密组织线：加入官方、民间、财团、教团或抵抗组织。
- 邪教/旧日传承线：利用禁忌知识和污染力量换取权力。
- 梦境线：在梦境都市、重复梦、梦中组织和现实裂缝中成长。
- 守夜人/反抗线：保护普通社会，对抗降临、污染和教团。
- 升格线：逐渐变成非人、神眷者、容器或类克苏鲁存在。
- 灭世/救世线：通过极端手段毁灭、重塑或拯救世界。

## 势力骨架

克苏鲁世界的势力通常围绕秘密、掩护、真相、污染和社会身份展开。

常见势力：

- 学校社团
- 网络怪谈圈
- 秘密研究小组
- 官方异常部门
- 财团实验机构
- 民俗/宗教组织
- 邪教细胞
- 抵抗组织
- 梦境共同体

创建势力需要默认硬条件，并额外看克苏鲁条件：隐蔽性、掩护身份、共同执念、禁忌知识来源、异常接触、保护措施、社会伪装、官方/邪教/实体注意风险。

势力成长权重偏向：秘密、理念、污染压力、伪装、内部执念、官方注意、背叛风险。

## NPC 生成方向

重要 NPC 可以包括父母、同学、老师、恋人、医生、记者、网络怪谈圈成员、官方联系人、邪教成员、研究员、梦境熟人、被污染亲友、隐藏保护者或伪装成普通人的非人存在。

普通 NPC 首先用正常人思维理解异常，除非他们已经是接触者、知情者、污染者或特殊组织成员。

## AI 续写原则

- 普通生活是真实路线，不是恐怖剧情前的填充物。
- 不要强迫玩家角色成为调查员。
- 异常不应立刻被所有人理解。
- 普通 NPC 可以知道怪事传闻，但不能自动知道旧日真相。
- 官方不是单纯正义或邪恶，它主要维持社会稳定、封锁真相、收容异常，也可能利用异常。
- 真相揭露越高，不一定越好。
- 社会正常度越低，越难回到普通人生。
- 污染/同化越高，可能变强，也可能失去人性。
- 不要把克苏鲁世界写成普通打怪升级。
- 不要让高颜值自动控制所有 NPC。
