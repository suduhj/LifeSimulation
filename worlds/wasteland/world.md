# 末日废土世界

## 世界定位

末日废土世界是 MVP 三个世界之一，用来承载崩坏世界中的长期人生模拟。它不是单纯打怪、刷装备或营地经营游戏，而是资源稀缺、亲密关系、疾病伤残、营地建设、阵营冲突、道德抉择和生命延续共同塑造的一生。

玩家角色可以出生在灾前末期、灾后庇护所、流民队伍、废墟营地、军阀控制区、商队、实验设施或荒原边缘聚落。年龄仍然是核心轴，生存天数和营地阶段是世界专属进度。

## 人生模拟重点

废土人生不只是活过下一天，也包括：

- 童年是否安全，谁保护玩家角色，谁牺牲。
- 饥饿、疾病、伤残、孤独、暴力和信任如何改变性格。
- 玩家角色是成为独行者、医生、领袖、掠夺者、工程师、商人、变异适应者，还是普通幸存者。
- 玩家角色能否建立家庭、保护同伴、创建营地、控制地盘或让文明重新萌芽。
- 末日选择会留下长期后果，而不是每次事件后清零。

## 世界专属进度

| 进度 | 含义 |
|---|---|
| 生存天数 | 玩家角色在废土秩序中的持续生存时间。 |
| 营地阶段 | 无营地、临时据点、稳定营地、武装聚落、区域势力等。 |
| 资源储备 | 食物、水、药品、弹药、燃料、工具、种子、零件。 |
| 伤病状态 | 感染、断肢、慢性病、疲劳、创伤、辐射或变异。 |
| 阵营信任 | 与营地、商队、帮派、庇护所、军阀、技术组织的信任。 |
| 环境危险 | 气候、污染、变异生物、掠夺者、瘟疫、资源枯竭。 |

## 核心路线

- 独行幸存者：少牵挂、自由高、风险高，依赖体质、运气、技能和谨慎。
- 营地建设者：收留成员、分配资源、建立制度、处理内斗和外敌。
- 商队/拾荒者：探索废墟、交易、运输、情报、黑市和高风险收益。
- 军阀/掠夺者：通过武力、恐惧、地盘和资源控制建立秩序。
- 医生/技术人员：用知识、维修、医疗、农业或能源重建生存基础。
- 变异适应者：身体被环境改变，可能变强、被排斥或失去人性。
- 庇护所统治者：在封闭资源体系中处理阶级、谎言、继承和叛乱。
- 荒原救世者：重建道路、联盟、水源、通信或新文明秩序。

## 势力骨架

废土势力通常围绕食物、水、防御、地盘、交通、医疗、武器和信任展开。

常见势力：

- 家庭小队
- 拾荒队
- 临时营地
- 庇护所
- 商队
- 民兵
- 掠夺者帮派
- 军阀领地
- 技术维修组织
- 医疗站
- 新文明议会

创建势力需要默认硬条件，并额外看废土条件：食物、水、武器、防御能力、医疗物资、地盘、交通、成员技能、可信领袖、让人活下去的证据。

势力成长权重偏向：经营建设、补给、防御、地盘、成员技能、生存信任、贸易路线和武装冲突。

## NPC 生成方向

重要 NPC 可以包括父母、兄弟姐妹、同伴、拾荒搭档、医生、机械师、营地领袖、掠夺者首领、商队联系人、背叛者、被救儿童、旧世界专家、变异者、庇护所继承人。

NPC 不应只是敌人或资源。废土 NPC 有恐惧、利益、亲情、忠诚、创伤、背叛、依赖和求生欲。

## AI 续写原则

- 生存压力来自长期稀缺，不只是战斗。
- 高运气可以带来转机，但不能消除饥饿、疾病、暴力和资源成本。
- 家境在末日中可表示出生庇护所、家族资源、旧世界遗产、装备、知识或人脉。
- 营地扩张必须有成本，可能带来饥荒、内斗、暴露、敌人和道德压力。
- 失败不一定立刻死亡，也可以造成伤残、资源损失、同伴死亡、信任崩塌或长期创伤。
- 不要把废土只写成打怪刷装备。
- 不要忽视疾病、疲劳、食物、水、信任和安全感。
