# 修仙世界

## 世界定位

修仙世界是 MVP 三个世界之一，用来承载从凡人生命到超凡生命层次跃迁的人生模拟。它不是整个游戏本体，也不能把修仙规则默认套到克苏鲁生活世界或末日废土世界。

这个世界的核心体验是：出生身份、灵根天赋、家族或宗门资源、凡人生活、拜师入门、修炼突破、道侣亲友、宗门政治、散修奇遇、家族兴衰、因果仇怨、寿命压力和飞升幻想，共同塑造玩家角色的一生。

## 人生模拟重点

修仙不是单纯升级。AI 写作时必须让修炼选择回到人生本身：

- 玩家角色如何出生、成长、拜师、离家、成名、衰老或超脱。
- 亲人、师父、同门、道侣、弟子、仇敌和宗门如何改变玩家角色的人生。
- 境界提升如何带来寿命、地位、资源、责任、孤独、因果和敌人。
- 玩家角色可以追求长生，也可以追求权力、亲情、复仇、自由、宗门、家族、魔道、隐居或飞升。

## 世界专属进度

修仙世界除年龄外，重点追踪：

| 进度 | 含义 |
|---|---|
| 境界 | 凡人、炼气、筑基、金丹、元婴、化神等生命层级。 |
| 修炼根基 | 灵根、功法适配、突破质量、道基稳定度。 |
| 宗门/家族地位 | 外门、内门、真传、长老、族人、继承人、客卿等身份。 |
| 资源 | 灵石、丹药、法器、洞府、功法、师承、秘境资格。 |
| 因果与声望 | 恩怨、承诺、仇家、名声、道心评价、正邪立场。 |
| 寿命压力 | 高境界延寿，低境界与伤病会带来时间压力。 |

MVP 境界：凡人、炼气、筑基、金丹、元婴、化神。

正式版扩展：炼虚、合体、大乘、渡劫、飞升。

## 核心路线

- 宗门路线：入门考核、外门挣扎、内门竞争、真传培养、师承派系、宗门大战。
- 散修路线：自由探索、遗迹、黑市、临时同盟、危险交易、无人庇护的突破。
- 家族/氏族路线：血脉、联姻、族产、继承权、家族兴衰、族内权斗。
- 魔修/异道路线：代价型力量、禁术、背叛、追杀、快速成长和不可逆污染。
- 隐修路线：避开争斗，追求长生、炼丹、阵法、游历、收徒或守护一地。

路线可以转换。例如家族子弟离家成为散修，散修被宗门招揽，宗门弟子被逐出后走向魔修或隐修。

## 势力骨架

修仙世界的势力通常围绕资源、传承、境界和名分展开。

常见势力：

- 宗门
- 修仙家族
- 散修联盟
- 商会
- 魔道组织
- 秘境传承支脉
- 王朝供奉体系

创建势力需要默认硬条件，并额外看修仙世界条件：个人实力、境界、传承、师承或家族背书、灵地资源、法器功法、名声、誓约和正邪立场。

势力成长权重偏向：理念/道统、资源、等级秩序、师徒关系、宗门政治、敌对势力冲突。

## NPC 生成方向

重要 NPC 可以包括父母、族老、师父、同门、竞争者、道侣、弟子、仇家、宗门高层、秘境守护者、魔修引路人、隐藏大能。

NPC 模板是身份锚点，不是五项全强模板。比如真传弟子代表宗门资源高，不代表颜值、智力、体质、运气全部优秀。

## AI 续写原则

- 境界突破是生命层次跃迁，不是小额属性提升。
- 凡人阶段仍然要有完整人生，不要急着跳到修炼爽点。
- 家境在修仙世界可表示宗门资源、师承、势力地位、法宝、洞府和灵石。
- 颜值可表示气质、道韵、生命层次和神性，不只是脸好看。
- 天骄会得到资源，也会引发嫉妒、争夺、夺舍、抢根骨、联姻或保护。
- 高境界人物不应自动五项全强，除非身份、天赋或剧情定位支持。
- 不要让所有路线都强行变成宗门路线。
- 不要把修仙规则套到其他世界。
